# 로컬 AI 노트 (local-learning-meeting-notes)

수업·회의 **전사문 → 학습/업무 산출물**로 변환하는 개인용 로컬 우선 AI 노트 PWA.
모든 데이터(전사문·분석결과·설정)는 **이 기기의 브라우저(IndexedDB)에만** 저장됩니다.

## 실행

```
node server.mjs
```

→ 브라우저에서 `http://localhost:4173`

## 사용 (M1)

1. **설정** 탭에서 분석 API 키 입력
   - 기본 공급자: Anthropic(Claude), 모델 `claude-opus-4-8` (저비용은 `claude-haiku-4-5`)
   - 키는 코드에 없고, 이 기기 IndexedDB에만 저장됨
2. **새 기록** 탭에서 모드(수업/회의) 선택 → 회의는 템플릿(일반/FDD/M&A/협상) 선택
3. 전사문 붙여넣기(또는 폰 키보드 마이크 받아쓰기) → **분석**
   - 회의 모드 + API 분석 시, 외부 전송 경고 후 동의해야 진행
   - 키 없으면 **수동 LLM 모드**로 자동 폴백(프롬프트 복사→외부 LLM→결과 붙여넣기)
   - API 호출/파싱 실패 시 **규칙 기반**으로 폴백
4. **기록 저장** → 기록 탭에서 목록/상세/삭제

## 어댑터 구조

- **AnalysisAdapter** (핵심): `ApiLlm`(기본) / `ManualLlm`(폴백) / `RuleBased`(최소 폴백)
- **TranscriptionAdapter**: `Manual`(동작) / `CloudStt`(M2) / `LocalWhisper`(M4)

## 개인정보

- 분석 시 외부로 나가는 것은 **전사 텍스트뿐**(사용자 동의 시). 녹음 오디오는 1단계에서 미전송.
- (M2) 클라우드 STT 도입 시 오디오 전송은 별도 동의 기반으로 추가 예정.

## 로드맵

- **M1 (완료):** 변환 엔진 — 수동/붙여넣기 전사문 → 5종 템플릿 분석 → 저장/목록/상세
- **M2:** 폰 현장 루프 — PWA 설치, 녹음(백업) + 클라우드 STT(route B), 받아쓰기 UX
- **M3:** Markdown 내보내기·JSON 백업/복원, 설정/오프라인 마감
- **M4:** 로컬 Whisper(WASM, PC 우선) — 녹음→자동 전사
